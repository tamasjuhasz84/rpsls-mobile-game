# Day 22 - Feature Graphic Brief (Play Store)

Datum: 2026-03-30
Statusz: Draft v1
Asset type: Google Play Feature Graphic
Target size: 1024 x 500 px (JPG or PNG)

## 1. Goal

Atadni 1 kepen:

- hogy ez RPSLS (5 jel)
- hogy tournament + survival + daily loop van
- hogy gyors, kompetitiv sessionelmenyt ad

## 2. Creative direction

Visual tone:

- Futuristic neon arena (blue-magenta palette)
- Kozepi fokusz: 5 hand signs + tactical arrows
- Energy beam / stage glow for momentum

Core message line options (max 1 rovid headline):

- HU: Gyors RPSLS Torna
- HU: Daily + Survival + Score
- EN: Fast RPSLS Tournament
- EN: Daily Challenge and Survival

## 3. Composition guideline

- Safe center area: kb 70% width
- Avoid dense text on edges (Play UI crop risk)
- Keep one clear focal point (middle symbol cluster)
- Reserve lower-right area for optional short CTA badge (if needed)

## 4. Typography guideline

- Strong geometric sans, high contrast
- 1 headline maximum
- Do not place small paragraph text
- Keep text readable at reduced thumbnail scale

## 5. Brand consistency

Must align with in-app style:

- Dark cosmic background
- Neon accent lights
- Hand icons / move identity

## 6. Export checklist

- [ ] 1024x500 exact
- [ ] RGB color space
- [ ] No transparency dependency
- [ ] No tiny unreadable text
- [ ] File size optimized for quick listing load

## 7. QA checklist before upload

- [ ] Thumbnail readability check (small preview)
- [ ] HU/EN headline version prepared (if text embedded)
- [ ] No legal-sensitive claims ("best", "#1")
- [ ] Visuals match final app screenshots (Day 23)
